#  Manual de Usuario — Editor de Texto con Tkinter

Este programa lo desarrollamos en **Python** utilizando la librería **Tkinter** para crear la interfaz gráfica.  
Trabajamos un **editor de texto simple** con opciones para abrir, guardar y editar.

---

##  Introducción

El programa permite al usuario **abrir archivos de texto existentes**, **editar su contenido** y **guardarlo nuevamente**, también esta la opción de guardar como que es para sobreescribir el archivo con otra extensión.  
Además, incluye una **sección de ayuda** con información del proyecto.

Fue desarrollado como un proyecto final del segundo ciclo de ingenieria en sistemas de la universidad Mariano Galvez por **Junior Alvaro René Barrios Flores** y **Maynor Suriel Del Valle Alvarado**.

---

##  Requisitos del Sistema

Para ejecutar correctamente este programa, necesitas:

- Python **3.8** o superior  
- Librería estándar `tkinter`
- Sistema operativo: **Windows**, **macOS** o **Linux**
